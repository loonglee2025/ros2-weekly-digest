### 2. ROSscope — Open Source Observability Platform for ROS 2 Fleets (v0.1.0)
- **Date:** 2026-06-07
- **URL:** https://discourse.openrobotics.org/t/rosscope-open-source-observability-platform-for-ros-2-fleets-v0-1-0/55338
- **Summary:**
  - Release of **ROSscope v0.1.0**, an open source monitoring platform for ROS 2 systems (described as "Prometheus + Grafana but with native ROS 2 awareness").
  - Features real-time topic monitoring (publish rate, message count, publisher health), automatic anomaly detection via z-score baselines, interactive publisher/subscriber graph mapping, service availability tracking, node lifecycle state tracking, and alert management.
  - Includes a pre-built Grafana dashboard auto-provisioned on first boot.
  - One-command install via curl or Docker (`docker pull jawadglitch/rosscope:latest`).
  - Roadmap includes multi-robot fleet support, persistent storage, and auto-remediation.
  - GitHub: https://github.com/jawad-glitch/ROSscope
